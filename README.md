# Hirameki Challenge

「ひらめきチャレンジ」は、クリック・タップだけで遊べる一人用のカジュアル脳トレ Web ゲームです。

## Features

- React + TypeScript + Tailwind CSS
- 6 種類のミニゲームをランダム出題
- スコア、コンボ、ノーミス、ランク判定
- ローカル保存の自己ベスト、実績、レベル、コイン
- デイリーミッション、ログインボーナス、ショップ風 UI
- PC、スマホ、タブレット対応

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
